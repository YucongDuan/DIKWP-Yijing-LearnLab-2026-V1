# DIKWP Yijing LearnLab 2026 V1

A standalone, offline-first learning prototype for Yijing / 易学 education. It supports classical text reading, 64-hexagram structure learning, DIKWP semantic interpretation, evidence-ledger study, teaching design, AI-use governance, and non-predictive reflective practice.

## Quick start
Open `index.html` in a browser. No server, database, model API, account, or network is required.

Optional CLI:
```bash
python tools/run_yijing_passport.py examples/sample_learning_note.json --out outputs
```

## Boundary
This system is for cultural, philosophical, textual, historical, and reflective learning. It is not a fortune-telling tool and must not be used for medical, legal, financial, psychological crisis, marriage-control, exam-admission, or other high-stakes decisions.

## Design lineage
The system follows the DIKWP ecosystem pattern: Evidence Ledger, Action/Interpretation Ticket, Human Review, Residual, Kill/Recovery, and Static Boundary Audit.
